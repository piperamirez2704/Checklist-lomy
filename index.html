⚙️ Conectar con Google Sheets

Para enviar datos necesitás configurar un Google Apps Script como puente. Solo se hace una vez:

  1. Abrí tu planilla "Checklist dinámico" en Google Sheets
  2. Andá a Extensiones → Apps Script
  3. Borrá el contenido y pegá el código que te pasamos
  4. Hacé clic en Implementar → Nueva implementación
  5. Tipo: Aplicación web · Acceso: Cualquier persona
  6. Copiá la URL de la aplicación web y pegala acá abajo
